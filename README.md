# Benchmarks

Benchmarks repository in the VNN-LIB format

## _ACAS XU_

Contains the aerial collision avoidance benchmarks

## _ACC_

Contains the benchmarks presented at ECMS 2022

## _Drones_ 

- 1d hovering: contains the benchmarks presented in the RCRA workshop 2022

## _RL_

Contains some RL case studies presented in VNNCOMP 2022

- Cartpole: the inverted pendulum problem
- Lunar Lander: the rocket trajectory optimization example
- Dubins Rejoin: the problem of an unmanned aerial vehicle following a manned one
